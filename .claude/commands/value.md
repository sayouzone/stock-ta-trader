Analyze a stock using the 5-stage Value Investing framework.

Usage: /value <TICKER> [--period 2y]

Steps:
1. Run `python main.py value <TICKER>` with the given arguments
2. Review the 5-stage analysis results
3. Provide a concise summary highlighting:
   - Overall grade and score
   - Key valuation metrics (PER, PBR, EV/EBITDA)
   - Profitability (ROE, margins, FCF)
   - Financial health concerns
   - Margin of safety
   - Entry timing assessment
4. Note any warnings or value trap risks
